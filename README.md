## Game Description
MineGrow is a resource-management game where players grow plants, harvest and sell them for profit, and use the earnings to upgrade the town and expand their mining base.

## Platform
- Windows

## Engine
- Unity

## How to Run the Game
1. Go to `Builds`
2. Double-click `MineGrow.exe`

## Controls
- WASD – Move
- Left Click – Interact
